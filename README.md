# 🏥 MedTrack – Patient Management System

MedTrack is a web-based patient management application designed to help healthcare providers efficiently manage patient records, appointments, and medical data in a simple and user-friendly interface.

---

## 🚀 Features

- 👤 Add, update, and delete patient records
- 📋 View complete patient history
- 📅 Appointment scheduling and tracking
- 🔍 Search and filter patients easily
- 💊 Manage diagnosis and treatment details
- 📊 Clean and responsive UI
